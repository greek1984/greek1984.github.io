Занимаюсь разработкой программных роботов (RPA) и автоматизацией бизнес-процессов на Python. Создаю надежные решения, которые заменяют ручной труд, избавляют ваших сотрудников от часов рутины и полностью исключают ошибки человеческого фактора.

Моя специализация:

✅ RPA (Роботизация): Создание ботов для автоматического ввода данных в 1С, CRM и закрытые корпоративные системы (эмуляция клавиатуры/мыши).
✅ Computer Vision (Компьютерное зрение): Настройка скриптов на визуальное распознавание интерфейсов (работают без привязки к внутреннему коду программ).
✅ Сбор данных (Web Scraping): Написание умных парсеров для сбора контактов, цен, товаров с любых площадок.
✅ Работа со сложным Web: Обход защит, использование Selenium для сайтов с динамической подгрузкой (JavaScript).

Примеры из портфолио

• RPA-бот для «1С: Бухгалтерия» (Computer Vision) — Автоматизация ввода первичной документации. Бот распознает интерфейс учетной системы, самостоятельно кликает по нужным полям и создает карточки контрагентов.

• Высоконадежный web-парсер — Извлечение данных из сложного многоуровневого динамического конфигуратора продукции.
